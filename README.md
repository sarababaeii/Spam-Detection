# Persian Spam Email Detector

An NLP project for detecting spam emails in Persian, using a k-Nearest Neighbours (kNN) classifier.

This project applies Natural Language Processing (NLP) techniques and probabilistic similarity measures to distinguish between spam and non-spam (ham) Persian emails.
It implements a k-Nearest Neighbours classifier from scratch, trains and evaluates the model using features derived from text similarity metrics, and compares the performance of cosine similarity and TF-IDF weighting.
